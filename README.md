# Campus Lost & Found System

A full-stack web application that helps students report and recover lost items on campus.

## Features

- Report lost items
- Report found items
- Search lost items
- User authentication
- Responsive UI
- Item details page

## Tech Stack

Frontend
- React
- Tailwind CSS
- React Router

Backend
- Spring Boot
- REST API
- JWT Authentication

Database
- MySQL / MariaDB
